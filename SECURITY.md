# Security

## Reporting a vulnerability

Please **do not** open a public issue for security-sensitive reports.

- Use [GitHub Security Advisories](https://docs.github.com/code-security/security-advisories/about-github-security-advisories) (**Security → Report a vulnerability**) if the feature is enabled on this repository, or
- Contact the maintainers privately with enough detail to reproduce and assess impact.

We will acknowledge receipt when possible and coordinate a fix and disclosure timeline.
